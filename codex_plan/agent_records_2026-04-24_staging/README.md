# Agent Records

本目录统一归档 agent 工作记录和对应讨论记录。

命名规则：

- `agent_work_YYYY-MM-DD/`：对应日期的 agent 工作交接、实现说明、风险提示。
- `discuss_YYYY-MM-DD/`：对应日期的问题与回答。

对话归档规则：

- 从 2026-04-21 开始，本对话框中的每轮用户需求都归档到当天的 `discuss_YYYY-MM-DD/`。
- 每轮需求生成一个 `questionN.md` 和对应的 `answerN.md`。
- `questionN.md` 直接复制用户原始需求，不改写、不润色。
- `answerN.md` 记录 agent 的分析、实际修改、验证结果和残留风险。
- 后续如果用户在同一天继续提问，按 `question2.md`、`answer2.md` 递增。
